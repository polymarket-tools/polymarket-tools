Polymarket Tools for n8n

Automate prediction market trading, whale tracking, and AI signals. Free, open source, 15 templates.

Install in 60 seconds

1

Open your n8n instance

Self-hosted or n8n Cloud. Both work.

2

Go to Settings > Community Nodes

Click Install in the community nodes section.

3

Search and install

Enter the package name and click install:

n8n-nodes-polymarket-tools
4

Start building workflows

Search for "Polymarket" in the node panel. Market data and whale tracking work immediately. No API key needed.
